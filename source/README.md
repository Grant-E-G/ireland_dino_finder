# Source Library

`source/pdf/` is for local paper PDFs and is intentionally ignored by git.

The README cites the papers currently expected here. Re-download them from the
listed DOI/arXiv URLs if this directory is empty after cloning.

Additional theory notes are in `docs/fractional_numerics_pitfalls.md`. The
fractional numerics PDFs cached here include papers on nonsmooth-data
convergence, short-memory failure, sum-of-exponentials stability, variable-order
issues, and fractional Laplacian boundary behavior.
